Phrase Conditionnel de type 2

Le conditionnel de type 2est utilisé pour parler d'une situation hypothétique, irréelle ou peu probable dans le présent ou le futur.


Structure du conditionnel de type 2
if + prétérit simple, would + base verbale
Exemple :
If I had a million dollars, I would travel the world (Si j'avais un million de dollars, je voyagerais autour du monde.)
If she were here, she would help us.(Si elle était ici, elle nous aiderait.)

Les phrases conditionnelles avancées (Types 2 et 3)

Apprenez comment utiliser les phrases conditionnelles de type 2 et 3 en anglais.

Type Forme Exemple
Conditionnel Type 2 if + prétérit, would + base verbale If I had a lot of money, I would travel the world.
Conditionnel Type 3 if + past perfect, would have + participe passé If you had studied harder, you would have passed the exam.
